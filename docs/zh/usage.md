# 使用指南

## 开始使用

当前支持Chrome及使用Chrome内核的浏览器如Edge等

## 注册账号

账号注册并不是强制的，无需注册也能使用。部分高级功能需要联网才能启用比如个人图形库、系统管理、团队管理及分享功能等。

![Signup](/images/signup.png)

- 需提供唯一账号名称、昵称及邮箱等
- 邮箱非常重要，需要邮箱验证通过才能注册成功
